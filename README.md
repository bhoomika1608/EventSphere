# 🎉 EventSphere

EventSphere is a modern full-stack event management and discovery platform that allows users to explore, create, and manage events seamlessly. It provides a personalized experience with authentication, event recommendations, and an intuitive dashboard.

---

## 🚀 Features

- 🔐 **User Authentication**
  - Login/Register using Email
  - Social Login (Google / Facebook)
- 📅 **Event Management**
  - Create, update, and delete events
  - Browse events by category, date, or location
- 🎯 **Personalized Recommendations**
  - Discover events tailored to user preferences
- 🧑‍💼 **Admin Panel**
  - Manage users and events
  - Role-based access control
- 📱 **Responsive UI**
  - Mobile-first design for all devices
- 🌐 **Social Integration**
  - Share events
  - Social login support
